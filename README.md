# 🎮 Coding Quest - เกมเรียนรู้การเขียนโปรแกรม

เกมสำหรับเด็กเรียนรู้พื้นฐานการเขียนโปรแกรม (Coding) ผ่านการควบคุมรถให้เดินทางผ่านตาราง 7×7 ช่องด้วยคำสั่งเคลื่อนที่

## 📋 ไฟล์ที่รวมอยู่

```
├── coding_game_advanced.html      # ไฟล์เกมหลัก (เปิดไฟล์นี้เพื่อเล่น)
├── manifest.json                  # ไฟล์เมตาดาต้าสำหรับ PWA
├── service-worker.js              # ไฟล์สำหรับทำให้เกมทำงานออฟไลน์
├── README.md                       # ไฟล์นี้
└── GUIDE_IPHONE.md               # คู่มืออย่างละเอียดสำหรับเล่นบน iPhone
```

## 🚀 วิธีเล่นด่วน

### วิธี 1: เปิดไฟล์โดยตรง (ง่ายที่สุด)
1. ดาวน์โหลดไฟล์ `coding_game_advanced.html`
2. ดับเบิลคลิกเพื่อเปิดในเบราวเซอร์
3. เล่นเลย!

### วิธี 2: เปิดในเบราวเซอร์
1. คลิกขวาที่ไฟล์ `coding_game_advanced.html`
2. เลือก "Open with" → เลือกเบราวเซอร์ (Chrome, Safari, Firefox เป็นต้น)

### วิธี 3: ใช้ Local Server (สำหรับ iPhone)
```bash
# บน Mac/Linux
python3 -m http.server 8000

# บน Windows
python -m http.server 8000

# หรือใช้ Node.js
npx http-server
```

จากนั้นเปิด `http://localhost:8000/coding_game_advanced.html`

## 🎮 วิธีเล่น

### Manual Mode (โหมดควบคุมตรง)
1. **กดปุ่มลูกศร** บนหน้าจอ
2. **กดแป้นพิมพ์** (Arrow Keys หรือ WASD)
3. **ลากเมาส์** (Swipe บน iPhone)

### Programming Mode (โหมดเขียนโปรแกรม)
1. เลือก "📝 Programming" แท็บ
2. **ลาก** ท่อนโปรแกรม (⬆⬇⬅➡) ลงในช่องคำสั่ง
3. กด **▶ Run** เพื่อรันโปรแกรม
4. ดูรถเดินตามโปรแกรม
5. กด **↻ Reset** เพื่อรีเซ็ต

## 🎯 เป้าหมายของเกม

- 🚗 เดินทางจากจุด **A** (เขียว) ไป **B** (แดง)
- ⭐ **เก็บดาว** ก่อนเข้าประตู
- 🚪 เข้าประตู (🚪) เพื่อผ่านด่าน
- 🌳 หลีกเลี่ยงอุปสรรค (ต้นไม้, ก้อนหิน, แม่น้ำ)

## 📱 เล่นบน iPhone

### วิธีเร็วที่สุด:
1. **Add to Home Screen**
   - เปิด Safari
   - ป้อน URL ของเกม
   - แตะ **Share** → **Add to Home Screen**
   - ตั้งชื่อ "Coding Quest"
   - แตะ **Add**

2. เล่นไม่จำกัด! 🎉

### ควบคุมบน iPhone:
- **Tap ปุ่ม** = ใช้นิ้วแตะปุ่มลูกศร
- **Swipe/Drag** = ลากนิ้วในทิศทาง

📖 ดูรายละเอียดเพิ่มเติมใน **GUIDE_IPHONE.md**

## 🎓 สิ่งที่เด็กจะเรียนรู้

1. **ทิศทาง** - เข้าใจเกี่ยวกับพิกัด (x, y)
2. **ลำดับคำสั่ง** - ความสำคัญของลำดับการทำงาน
3. **ปัญหาแก้ปัญหา** - คิดแบบลำดับตรรกะ
4. **การทดสอบ** - ทดสอบและแก้ไขโปรแกรม
5. **ความยืดหยุ่น** - ลองวิธีต่างๆ
